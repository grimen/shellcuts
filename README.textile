h1. SHELLCUTS

Convenient shortcuts directly within the shell. More to come...

h2. Installation

<pre>sudo gem install grimen-shellcuts</pre>

h2. Usage

<pre>define [MAX_RESULTS] QUERY</pre>

h2. Examples

Input:

<pre>define "Big L"</pre>

or

<pre>define Big L</pre>

Output:

<pre>Definitions of "Big L" on the Web:
  
  * Lamont Coleman (May 30, 1974 - February 15, 1999), also known by his stage name Big L, was an American rapper.en.wikipedia.org/wiki/Big_L</pre>
